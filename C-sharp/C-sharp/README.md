# jieba - quickstart (C\# ver.)

## System

* Windows8
* .NET Framework 4.5

<br>

## Launch
以下以 Visual Studio Community 2015 作為 IDE 示範

**開啟專案**
```
檔案 > 開啟 > 專案/方案
選擇 path/to/C-sharp/C-sharp.csproj
```

**安裝套件**

```sh
工具 > NuGet套件管理員 > 套件管理器主控台
# 在主控台中書輸入以下指令
PM> Install-Package jieba.NET
執行專案
```
